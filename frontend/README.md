# GeoMatch Frontend

Simple React (Vite) UI to upload CSV/SQL, visualize results, and export.

## Setup

```powershell
cd frontend
npm install
# set the backend URL (default example below)
$env:VITE_API_URL = "http://localhost:8000"
npm run dev
```

Open http://localhost:5173 and set `VITE_API_URL` via your shell profile or a `.env` file supported by Vite.
